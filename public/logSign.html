SignUp | Login

A login page typically serves as a gateway for users to access a secure account or system. It requires users to enter their credentials, such as username and password.
A signup page, on the other hand, enables new users to create accounts for the service or platform. It usually involves filling out a form with necessary information like username, email address, and password, and sometimes additional details depending on the service.
Currently it is created using html and css so there is no JS validation and all.
