Typing Speed Game

A typing test is designed to find how fast one types in a given amount of time. We will be designing a typing game using Pure JavaScript CSS and HTML that presents a simple typing challenge and finds the performance of typing and the accuracy of the typed characters.
There is 3 level Easy,Medium and Hard.

The game shows a series of quotes that have to be typed in a specified time limit, as fast as possible.
Tech Stack : HTML, CSS, JS
