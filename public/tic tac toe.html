Tic Tac Toe

This is regarding Tic Tac Toe Game. Player starts playing the game and both the players make their moves in consecutive turns. The player who makes a 3-block chain wins the game. This game is built on the front-end using simple logic and validation checks only.

This game is purely made using HTML CSS and Javascript in a simple manner.
Tech Stack : HTML, CSS, JS
